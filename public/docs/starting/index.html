# Getting Started When you're ready to try Scrupal, everything you need to know is in this section. # Installation Installation of Scrupal is straight forward as it is packaged for your operating system. Use one of these links depending on which kind of system you want to install Scrupal on: Mac OS/X, Linux, Windows, Others.

Mac OS/X

The Scrupal installation comes as a standard DMG package. Download it from here and then follow these steps: + In Finder, browse to where you downloaded the `scrupal-x.y.dmg` file (x and y are digits). + Double click on the scrupal-x.y.dmg file to mount it. Notice the new disk icon on your desktop. + Double click the `scrupal-x.y` disk drive icon on your desktop. + Open the `bin` directory and double click on the `scrupal` script to get Scrupal running. + You can install Scrupal to a more permanent location by just copying the files to another location
